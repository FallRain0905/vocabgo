## 🚀 快速开始

### 1. 启动VocabGo RPA

```bash
cd scripts
python translator_gui.py
```

首次启动时，程序会自动检查Tesseract状态：
- 如果Tesseract未找到：显示依赖管理界面
- 如果Tesseract已安装：直接启动主程序

### 2. 使用OCR识别

#### 方法一：快捷键（推荐）

**操作步骤：**
1. 打开要识别的英语题目（确保文字清晰可见）
2. 按 `F8` 键（或自定义的快捷键）
3. 等待识别完成（通常1-3秒）

**功能特点：**
- ⚡ 自动截图并识别
- 📊 显示置信度和统计信息
- 🎯 高准确率（测试显示76.5%置信度）

#### 方法二：GUI按钮

**操作步骤：**
1. 在GUI中点击"📷 截图识别"按钮
2. 查看识别结果和统计信息
3. 如需翻译，点击"🧠 翻译识别内容"

**显示信息：**
- 识别的文本内容
- 置信度（百分比）
- 字符/行/段落/块数量
- 检测到的单词数

### 3. 查看识别结果

#### 结果显示区域

**"🔍 文本识别"区域显示：**
```
识别文本：
Hello 世界! This is a test. 这是一段测试文字。

统计信息：
- 平均置信度：76.5%
- 字符数：1968
- 行数：30
- 段落数：5
- 文本块数：17
- 检测单词数：412
```

### 4. 翻译识别内容

#### 翻译操作

**操作步骤：**
1. 在"🔍 文本识别"区域中查看识别的文本
2. 点击"🧠 翻译识别内容"按钮
3. 查看翻译结果

**翻译流程：**
1. 自动提取英语单词
2. 调用通义千问API进行翻译
3. 在"🇺🇸 中文翻译"区域显示结果
4. 使用翻译缓存提高速度

### 5. 配置OCR参数

#### 打开设置窗口

点击GUI中的"⚙️ 设置"按钮，找到"📷 OCR功能设置"部分。

#### 可配置参数

| 参数 | 说明 | 默认值 | 推荐值 |
|------|------|--------|--------|
| **识别语言** | OCR使用的语言 | chi_sim+eng | 英语题目: eng<br/>中英混合: chi_sim+eng |
| **快捷键** | 截图识别的快捷键 | F8 | F8, F9, Ctrl+F8 |
| **识别区域** | 自定义截图区域 | 全屏 | 精确区域提高准确率 |
| **启用OCR** | 是否启用OCR功能 | true | true |

#### 高级配置（可编辑配置文件）

在 `config/qwen-config.json` 中添加：

```json
{
  "ocr_enabled": true,
  "ocr_hotkey": "f8",
  "ocr_lang": "chi_sim+eng",
  "ocr_region": {
    "top": 200,
    "left": 100,
    "width": 1000,
    "height": 500
  }
}
```

## 🎯 使用技巧

### 提高识别准确率

#### 1. 优化屏幕设置
```
推荐配置：
- 显示缩放：100%（重要！）
- 分辨率：1920x1080或更高
- 字体大小：中等或较大
- 对比度：高对比度配色方案
- 背景颜色：浅色或白色背景
```

#### 2. 精确定位识别区域
```
方法一：使用预设区域
1. 将题目放在屏幕固定位置
2. 在设置中配置识别区域（top, left, width, height）
3. 使用快捷键识别，避免干扰元素

方法二：手动选择区域
1. 保留识别区域为空（全屏识别）
2. 使用快捷键识别完整屏幕
3. OCR引擎自动过滤干扰元素
```

#### 3. 优化识别参数

```python
# 在代码中调整参数（高级用户）
engine.set_psm(PSM.SINGLE_BLOCK)  # 单块模式更快
engine.set_oem(OEM.DEFAULT)             # 默认模式最准确
engine.set_language("eng")               # 纯英语题目更快
```

### 不同题型的处理

#### 听力选择题（题型2）
```
推荐流程：

2. 音频播放后自动识别并翻译

```
推荐流程：

2. 查看识别结果和置信度

4. 根据翻译结果选择答案

#### 选词填空题
```
推荐流程：
1. 按F8截图识别整个题目区域
2. OCR自动提取候选词汇和空格位置
3. 翻译候选词汇了解含义
4. 根据句意和词汇含义选择答案
```

#### 混合题型
```
推荐流程：
1. 听力部分：使用语音识别
2. 文本部分：使用OCR识别
3. 结合两种识别结果完成题目
4. 使用翻译功能辅助理解
```

## 🔧 故障排除

### OCR识别失败

#### 问题：按快捷键无反应
```
解决方案：
1. 检查快捷键是否与其他程序冲突
2. 尝试更换快捷键
3. 确保GUI窗口处于活动状态
4. 使用"📷 截图识别"按钮代替
```

#### 问题：识别结果为空
```
解决方案：
1. 检查屏幕是否有清晰文字
2. 确保题目区域没有被其他窗口遮挡
3. 调整识别区域参数
4. 检查文字清晰度和对比度
```

#### 问题：识别准确率低
```
解决方案：
1. 优化屏幕设置（100%缩放，高对比度）
2. 调整识别区域，避免干扰元素
3. 尝试不同的语言设置（eng vs chi_sim+eng）
4. 调整PSM参数（SINGLE_BLOCK vs AUTO）
```

#### 问题：识别速度慢
```
解决方案：
1. 减小识别区域
2. 调整OEM参数（TESSERACT_ONLY更快）
3. 关闭图像预处理（在代码中设置preprocess=False）
4. 使用更快的PSM模式
```

## 🎨 高级功能

### 详细OCR信息

新的OCR引擎提供了丰富的识别信息：

```python
# 在代码中可以访问详细结果
result = ocr_engine.recognize(screenshot)

# 文本信息
print(f"文本: {result.text}")
print(f"字符数: {result.num_chars}")
print(f"行数: {result.num_lines}")
print(f"段落数: {result.num_pars}")
print(f"块数: {result.num_blocks}")

# 置信度信息
print(f"平均置信度: {result.mean_conf}%")

# 单词级信息
for word in result.words:
    text = word['text']
    confidence = word['conf']
    x, y = word['left'], word['top']
    print(f"单词: {text}, 置信度: {confidence}%, 位置: ({x}, {y})")
```
