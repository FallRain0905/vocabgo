- ✅ 鼠标拖动选择屏幕区域

## ⚙️ 配置百度 OCR

### 1. 获取百度 OCR API Key

如果还没有百度 API Key，需要先申请：

1. **访问百度智能云控制台**
   - URL: https://cloud.baidu.com/
   - 使用百度账号登录

2. **开通文字识别服务**
   - 搜索"文字识别"
   - 点击"立即使用"
   - 选择"通用文字识别（标准版）"
   - 开通服务（新用户有免费额度）

3. **创建应用**
   - 在"文字识别"控制台中，点击"创建应用"
   - 填写应用名称：VocabGo 翻译助手
   - 填写应用描述：大学英语作业辅助

4. **获取 API Key**
   - 创建完成后，在应用列表中找到你的应用
   - 复制 **API Key** 和 **Secret Key**

### 2. 更新配置文件

编辑文件：
```
c:\Users\surface\OneDrive\Desktop\vocabgo\vocabgo-rpa\config\qwen-config.json
```

确保已有以下配置：

```json
{
  "baidu_api_key": "你的API Key",
  "baidu_secret_key": "你的Secret Key",
  ...
}
```

**说明**：
- 如果已经配置了百度语音识别，可以复用相同的 API Key 和 Secret Key
- 百度 OCR 和语音识别使用相同的认证系统

## 📱 使用方法

### 启动 GUI

```bash
cd scripts
start_gui.bat
```

### OCR 识别步骤

1. **打开微信网页**（app.vocabgo.com）
2. **切换到题目页面**（题型1：画线单词翻译）
3. **点击"🖼️ OCR"按钮** - 打开 OCR 窗口
4. **点击"📐 选择区域"按钮** - 打开全屏选择窗口
5. **鼠标拖动** - 在题目文字上拖动鼠标框选区域
6. **回到 OCR 窗口** - 自动恢复
7. **按 `~` 键** - 触发 OCR 识别
8. **查看结果** - 识别文字 + 翻译

### 快捷键说明

| 快捷键 | 功能 |
|--------|------|
| `~` | 触发 OCR 识别 |
| ESC | 取消区域选择 |

## ⚙️ 配置选项

### 修改快捷键

默认使用 `~` 键，如需修改，编辑 `translator_gui.py` 第 820 行左右：

```python
keyboard.add_hotkey('`', self.trigger_ocr)  # 修改这里的键位
```

可用键位：`F1`-`F12`、`Ctrl+A`、`Ctrl+Shift+Q` 等

### 修改窗口透明度

编辑 `translator_gui.py` 第 890 行左右：

```python
self.window.attributes('-alpha', 0.95)  # 修改透明度 (0.0-1.0)
```

### 修改选择窗口透明度

编辑 `translator_gui.py` 第 920 行左右：

self.selection_window.attributes('-alpha', 0.3)  # 修改透明度 (0.0-1.0)

## ❓ 常见问题

### Q: 快捷键 `~` 没有反应？
A:
1. 检查是否正确安装了 `keyboard` 包
2. 检查是否正确选择了识别区域
3. 查看终端是否有错误信息
4. 尝试重新打开 OCR 窗口

### Q: 提示"OCR 不可用"？
A:
1. 检查配置文件中的百度 API Key 和 Secret Key
2. 确认已开通百度文字识别服务
3. 检查网络连接
4. 查看终端的错误信息

### Q: 选择区域时窗口闪烁？
A: 这是正常现象，系统在切换窗口焦点。

### Q: 识别结果不准确？
A:
1. 提高屏幕分辨率
2. 使用 100% 缩放
3. 确保文字清晰
4. 减少选择区域的空白部分

### Q: 快捷键与其他软件冲突？
A: 修改快捷键（见"配置选项"部分）。

### Q: 百度 OCR 有费用吗？
A:
- 新用户有免费额度（通用文字识别标准版）
- 免费额度用完后按量计费
- 通用文字识别：约 ¥0.0025/次
- 估算：100 次识别 = ¥0.25
- 价格很低，日常使用几乎免费
